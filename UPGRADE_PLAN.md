# 月影车姬 · 功能升级规划

> 本文档整理了月影车姬 (YueYingCheJiBot) 下一阶段的功能升级建议，供开发者按优先级逐步落地。

---

## 1. AI 与核心匹配升级

### 1.1 向量语义搜索 (RAG + pgvector)

- 在现有 PostgreSQL 中启用 `pgvector` 扩展（`CREATE EXTENSION IF NOT EXISTS vector;`），无需引入新数据库服务。
- 将灯笼描述（`description`、`type`、`city` 等字段）在投稿审核通过时异步生成 1536 维向量，存入新增的 `embedding vector(1536)` 列。
- 用户输入自然语言（如"温柔、信义区附近"）时，由 AI 将其转为查询向量，在数据库中执行近似近邻（ANN）搜索：
  ```sql
  SELECT * FROM lanterns ORDER BY embedding <=> $1 LIMIT 5;
  ```
- 可显著提升"月影媒婆 AI"的语义匹配精准度，替代现有关键词模板匹配。

### 1.2 AI 多模态鉴真升级

- **视频鉴真**：接受短视频投稿，调用多模态模型（如 Grok Vision / 通义千问 VL）逐帧抽样鉴别真实性。
- **pHash 网图黑名单库**：对每张通过审核的图片计算感知哈希（pHash），存入 `image_hashes` 表。新投稿图片先与库中哈希做汉明距离比较，命中则直接拒绝并扣除兰花令，从源头杜绝重复假图。
- **AI Persona 切换**：为"媒婆"设计多套回复风格（御姐音 / 高冷风 / 元气风），用户可在 `/settings` 中选择，丰富互动体验。

## 2. Mini App 交互升级

### 2.1 地图聚合聚类 (Marker Clustering)

- 引入 `Leaflet.markercluster` 插件（CDN 方式，无需打包），当地图缩小时自动将密集标记合并成数字气泡，放大后散开。
- 已在 `mini_app.html` 中完成实现（见代码变更）。

### 2.2 动态路线导航

- 在灯笼弹出框中增加"导航"按钮，根据用户地理位置（`navigator.geolocation`）计算到灯笼模糊位置的直线距离，并提供跳转到高德 / Google Maps 的导航链接：
  ```
  https://maps.google.com/?q={lat},{lng}
  ```
- 面向大陆用户可优先接入高德 Web API。

### 2.3 骨架屏 / PWA / 深色模式

- 利用 Telegram Mini App SDK 的 `Telegram.WebApp.themeParams` 自动获取客户端主题色，实现深色/浅色模式无缝切换。
- 为地图加载前展示骨架屏（灰色占位区块），减少白屏感知时长。
- 添加 `manifest.json` 和 Service Worker，使 Mini App 支持 PWA 离线缓存。

## 3. 兰花信用生态与游戏化

### 3.1 等级与成就系统

在现有兰花令信用分基础上，新增用户等级：

| 等级 | 所需兰花令 | 专属特权 |
|------|-----------|---------|
| 🌱 寻花客 | 0 – 99 | 每日 5 次 AI 查询 |
| 🌸 摘星人 | 100 – 499 | 每日 15 次查询 + 解锁高分灯笼 |
| 🌙 月影传奇 | 500+ | 无限查询 + 隐藏极品灯笼 + 专属徽章 |

- 在 `credit.py` 中新增 `get_level(credit: int) -> str` 函数，返回等级及对应权限。
- `/start` 菜单及 Mini App 状态栏显示当前等级徽章。

### 3.2 匿名评价与打分反馈

- 用户体验后，通过 Inline Keyboard 在 Telegram 内部提交匿名标签式评价：
  `🌟🌟🌟🌟🌟 极品` / `📸 照骗` / `🤝 服务佳` / `⚠️ 中介勿扰`
- 评价结果反哺至灯笼的 `authenticity_score`（加权滑动平均），动态调整 AI 推荐权重。
- 超过阈值的负评自动触发"待复核"状态，通知管理员。

## 4. 群管与流量反哺（吸引群主）

### 4.1 AI 自动打假拦截

- 机器人作为群管理员时，监听群内图片消息（`content_types=['photo']`），异步调用 `ai.py` 中的鉴真接口。
- 识别为高概率假图（概率 ≥ 85%）时，自动回复：
  > ⚠️ 月影守护提示：经 AI 鉴定，此图片为网图/假图概率 {score}%，请群友注意防骗。
- 支持群主通过 `/setup` 开关此功能。

### 4.2 群主特权（月影阁主身份）

- 机器人被添加为群管理员时，自动识别群主 Telegram ID，在 `group_configs` 表中赋予 `is_admin_vip = True`。
- 阁主专属特权：
  - 初始赠送 200 兰花令。
  - 解锁无限 AI 媒婆搜索配额。
  - 一票否决：可通过 `/ban_lantern <id>` 直接下架群内劣质资源。
  - 入群欢迎语个性化定制（`/setwelcome <文案>`）。

### 4.3 群内盲盒灯笼掉落

- 群主通过 `/setup` 配置每日定时掉落（如每天 21:00）或按消息量触发（群内消息满 500 条）。
- 触发后，机器人在群内发送一张模糊处理的"极品灯笼"卡片，群友发送口令（默认 `月下寻花`）可解锁详情。
- 口令响应使用 `APScheduler` 或 aiogram 的 `Message` 过滤器实现，无需外部队列。

### 4.4 月影联盟 · 双向引流

- Mini App 主菜单新增"认证交流群"板块，展示已接入机器人的优质群组列表（群名、人数、二维码 / 邀请链接）。
- 准入规则：群主将机器人设为管理员 + 群人数 ≥ 100，即可自动上榜。
- 灯笼资源展示时带上来源群组标签，增加群组曝光度，实现双向引流闭环。

### 4.5 云端黑名单共享库

- 建立 `blacklist` 表（`tg_id BIGINT PRIMARY KEY, reason TEXT, reported_by BIGINT`）。
- 群主通过 `/report @用户` 或 `/report <tg_id>` 上报黑名单候选，管理员审核通过后自动同步至所有接入群组。
- 举报通过奖励群主 50 兰花令，激励社区共治。

## 5. Telegram 生态深度集成

### 5.1 Inline Mode（内联分享卡片）

- 在 `@BotFather` 中开启 Inline Mode（`/setinline`）。
- 用户在任意聊天框输入 `@YueYingCheJiBot 台北 大学生`，机器人返回模糊处理过的灯笼卡片列表，点击即可分享给好友，实现裂变传播。
- 使用 aiogram 的 `InlineQuery` 处理器实现，查询结果以 `InlineQueryResultArticle` 形式返回。

### 5.2 社区传奇频道推送

- 当灯笼 `authenticity_score ≥ 95` 且收藏数 ≥ 20 时，自动将格式化的图文卡片（含模糊图片 + 评分 + Inline Button）推送到官方 Telegram Channel。
- 使用 `APScheduler` 每小时检查一次，避免重复推送（新增 `is_featured` 布尔字段）。

### 5.3 月影提醒订阅（APScheduler）

- 允许用户订阅特定城市 + 类型组合（`/subscribe 台北 KH`），新灯笼上架时自动推送私信通知。
- 订阅关系存储在 `subscriptions` 表，`APScheduler` 每 30 分钟轮询新增灯笼并批量推送。

## 实施优先级建议

| 优先级 | 功能 | 理由 |
|--------|------|------|
| ⭐⭐⭐ | AI 自动打假 + 群主特权 | 最快拉新群主，短期内扩大流量盘 |
| ⭐⭐⭐ | pHash 网图黑名单 | 直接提升平台内容质量，减少人工审核负担 |
| ⭐⭐ | pgvector 语义搜索 | 提升 AI 媒婆核心体验，技术壁垒最高 |
| ⭐⭐ | 等级系统 + 匿名评价 | 提升用户留存和平台数据质量 |
| ⭐⭐ | 月影联盟双向引流 | 群主最关注"涨粉"，引流闭环是长期增长引擎 |
| ⭐ | Inline Mode 分享卡片 | 裂变效果强，但需要一定的用户基数才能显现 |
| ⭐ | Telegram 提醒订阅 | 提升用户活跃度，但开发量偏大 |
